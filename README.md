# Interview Monitoring System Using Image and Audio Processing

This project monitors online interviews using real-time face detection, audio detection, and movement tracking.

## Features
- Multiple Face Detection
- Multiple Voice Detection
- Face Movement Tracking
- Live Monitoring
- Final Report
- Excel Report Generation
- GUI Interface

## Tools Used
Python, OpenCV, NumPy, SoundDevice

## How to Run
pip install -r requirements.txt  
python gui.py
